ghc-mod-5.2.0.0: Happy Haskell Programming

The ghc-mod command is a backend command to enrich Haskell programming on editors including Emacs, Vim, and Sublime. The ghc-mod command is based on ghc-mod library which is a wrapper of GHC API. This package includes the ghc-mod command, the ghc-mod library, and Emacs front-end (for historical reasons). For more information, please see its home page.
